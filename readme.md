# Veteriner Otomasyonu
Bu proje, bir veteriner kliniğinde hayvan kayıtlarının tutulmasını sağlayan
menü tabanlı bir **Java** uygulamasıdır.

Kullanıcılar ana menü üzerinden kayıt ekleme işlemi yapabilir.
Girilen hayvan adı, tür, yaş ve sahip bilgileri dizilerde saklanmaktadır.

## Kullanılan Teknolojiler
- Java
- Scanner
- Diziler
- Metodlar
- while döngüsü
